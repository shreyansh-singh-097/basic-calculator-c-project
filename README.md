# 🧮 Basic Calculator in C

This is a simple command-line calculator program written in C. It performs basic arithmetic operations like addition, subtraction, multiplication, and division.

## 🚀 Features
- Addition
- Subtraction
- Multiplication
- Division
- Simple and easy to use
- Runs in terminal

 ## 🛠️ Tech Used
- C Programming Language
- Standard Library (stdio.h)

## ▶️ How to Run

1. Compile the code:
   gcc calculator.c -o calculator

2. Run the program:
   ./calculator

## 💡 How It Works
- User selects an operation
- Enters two numbers
- Program performs the calculation
- Displays the result

## 📂 File Structure
calculator.c  → main source code

## 📌 Example
Enter choice: 1  
Enter numbers: 5 3  
Output: 8
